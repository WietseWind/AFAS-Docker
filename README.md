# AFAS-Docker

Docker Sample for AFAS (usage: file-upload environment)

## Environment

- Ubuntu 14.04 LTS CLI
- Apache 2.4
- PHP 7.0 + OpCache + Mongo-driver
- Webroot: `/var/www/nodum_projects/default`
- `index.php` is placed in the webroot.

## Sample scripts

#### Building the image 

``` ./generate-image ```

#### Creating the container

``` ./generate-container ```
